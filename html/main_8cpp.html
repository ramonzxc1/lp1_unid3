Detailed Description

Arquivo principal do projeto empresa da unidade 3 de Linguagem de Programação 1.

O projeto consiste na simulação de uma empresa que possui um dono e seus funcionários(ASG, vendedores e gerentes). O código foi desenvolvido em C++ com orientação à objetos e utiliza classes(também tendo abstratas), herança e manipulação de arquivos. Na manipulação de arquivos, os dados são lidos e escritos em arquivos .txt. Também existem informações geradas no terminal, mas nada é lido a partir do mesmo.
